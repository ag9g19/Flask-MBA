Market Basket Analysis

Get Insights about you business

Market Basket Analysis is a widely-used technique to uncover associations between items and purchasing patterns in customers. It is used to produce product recommendations and marketing ideas by translating vast amount of customer transaction data into association rules.

This platform gives you the opportunity to get personalised MB-analysis on your transactional data as well as important insights and statistics. It will make all of your transactions data into a readable and useful format, that can help you improve your marketing strategy.

Some use cases include:

  1. Building Netflix-style recommendation engine
  2. Improve product recommendation on an e-commerce store
  3. Cross-sell products in retail setting
  4. Improve inventory management
  5. Increase sales
  6. Optimise store layouts
  7. Design product bundles

You must follow the instruction for the format of the data:

3 columns:

  1. Transaction ID
  2. Items
  3. Date

The first column of the dataset contains the transaction ID that the item in second column is part of. Hence, the dataset has to be ordered by each item purchased, not by each transaction. The third column shows the specific time of the corresponding transaction. It was added as a feature for the dataset to contain, because it provides useful information, that can result in more broad range of statistics.
